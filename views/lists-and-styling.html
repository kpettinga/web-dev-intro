Lists


Ordered Lists

Ordered lists use numbers. Use the <ol> tag to create Ordered Lists.

  1. list item
  2. list item
  3. list item
<ol>
	<li>list item</li>
	<li>list item</li>
	<li>list item</li>
</ol>

Unordered Lists

Unordered lists used bullets. Use the <ul> tag to create Unordered Lists.

  • list item
  • list item
  • list item
<ul>
	<li>list item</li>
	<li>list item</li>
	<li>list item</li>
</ul>

Nesting Lists

It's even possible to nest lists within lists. Here's an example of an unordered list inside of an ordered list.

  1. list item
    • list item
    • list item
    • list item
  2. list item
  3. list item
<ol>
	<li>list item
		<ul>
			<li>list item</li>
			<li>list item</li>
			<li>list item</li>
		</ul>
	</li>
	<li>list item</li>
	<li>list item</li>
</ol>


Styling the fonts

With the style attribute, we can change the font, color, and size of our text


Font Size

Use font-size to change the size of our text.

I'm a big paragraph!

<p style="font-size: 20px">I'm a big paragraph!</p>

Font Color

Use color to change the color of our text.

I'm a red paragraph!

<p style="color: red">I'm a red paragraph</p>

Font Family

Use font-family to change the font of our text. Some common fonts to choose from are:

Georgia
Verdana
Tahoma
Monaco
Arial

This paragraph uses the Georgia font.

<p style="font-family: Georgia">This paragarph uses the Georgia font</p>


More Styling!


Background Color

  • List item
  • List item
  • List item
<ul style="background-color: lightblue">
	<li>List item</li>
	<li>List item</li>
	<li>List item</li>

Aligning Text

Use text-align to change the alignment of text. You an choose from left, center, or right.

This is a right aligned paragraph.

<p style="text-align: right">This is a right aligned paragraph.</p>

Strong words

Use the <strong> tags to make text bold.

These are some strong words!

<p>These are <strong>some strong</strong> words!</p>

Emphasized words

Use the <em> tags to make text italics.

All I want is some italic text!

<p><em>All I want</em> is some italic text!<p>


Challenge


Create an html page with the following elements:

  1. Make an H1 heading at the top of your page.
  2. Create a paragraph under the H1 with any text you like.
  3. Create 1 ordered list with 3 list items inside of it.
  4. Create 1 unordered list with 3 list items inside of it.
  5. Put the unordered list inside one of the ordered list items.
  6. Style your text using at least 3 of the following properties and tags:
    • font-size
    • font-family
    • color
    • text-align
    • <strong></strong>
    • <em></em>
  7. Finally, change the background color of the body to lightblue.
    Hint: the style attribute will go on the body tag.
